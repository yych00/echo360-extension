# Echo360 CC 字幕助手

Echo360 CC 字幕助手是一款专为 Echo360 视频录播平台打造的浏览器扩展，提供实时双语字幕、智能断句、快捷键导航及字幕导出等功能，大幅提升学习体验。

## ✨ 核心功能亮点

- **实时双语字幕与翻译**：自动获取视频 CC 字幕，并支持多语言即时机器翻译，默认提供中英双语对照显示。
- **智能断句与平滑渲染**：搭载智能语句黏连引擎，将碎句合并为连贯的长句，并拥有防闪烁机制，阅读更流畅。
- **快捷键精准导航**：鼠标点击视频/字幕区域后，可通过键盘 `←` 和 `→` 方向键，**按字幕句子**的颗粒度精准快退和快进。
- **高度可定制界面**：支持自由调整字幕字号、中文/英文独立颜色及背景透明度，修改后免刷新即刻生效。
- **一键字幕导出**：支持将整节课的字幕以及翻译内容一键导出，非常方便期末复习与整理笔记。
